Studio Art Pages
What are Website Templates?

These templates are just HTML, CSS and JavaScript files that have been constructed as templates that are relatively easy for artists to understand, customize and fill in with their own content. Again this is a bit technical but not impossible to use for artists that don't want to wait for the official launch of the fully developed Art Pages beta platform.

Are these Templates Art Pages?

These templates are not the Studio Art Pages project. These templates are made available for artists that want to start building a website before we officially launch the beta version of the Art Pages platform. Getting your website running with these templates involves some basic HTML, CSS, JavaScript and hosting work.

How Do I Use a Template?

To use a tmeplate click it's download link and you will be taken to the Studio Art Pages GitHub. Here you'll be able to download the HTML, CSS and JavaScript files associated with the template to your computer where you can edit them and add your own content. You can find instructions for organizing your files for use with these templates at the file organization link.

What is GitHub?

GitHub is platform used by almost all professional web developers to share and edit code. In addition to a number of other features it provides free hosting for websites with the option of using a standard custom domain name (ie www.yourname.com). This provides a fantastic oppourtunity for artists who are willing to learn to use GitHub, allowing them to easily host and update professional websites for free if they are willing to do a little work learning the basic technical work necessary.
